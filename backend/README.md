# EnergyGrid Backend Server

Express.js backend API for the EnergyGrid Dashboard.

## Setup

```bash
npm install
npm start
```

Server runs on http://localhost:5000

## API Endpoints

- `GET /api/status` - Aggregation status
- `POST /api/aggregate` - Start aggregation
- `GET /api/results` - Get results
- `GET /api/stats` - Get statistics
- `GET /health` - Health check
